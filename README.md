# IM-BEE-SCM
 IM-BEE-SCM (Identifier Mass Spectrometry Batch-Effect-Eliminated Single-Cell Metabolomics) is a batch-effect-free approach that employs tandem laser-induced fluorescence (LIF) and mass spectrometry (MS).
